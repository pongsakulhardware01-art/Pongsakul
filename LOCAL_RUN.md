# วิธีการติดตั้งและรันโปรแกรมบนคอมพิวเตอร์ส่วนตัวของคุณ (Local Run Guide)

แอปพลิเคชันนี้เขียนด้วย **React (Vite) + Tailwind CSS** ซึ่งสามารถนำไปรันบนคอมพิวเตอร์ของคุณเองได้ง่ายๆ ภายในไม่กี่วินาที โดยมีหลากหลายวิธีที่สะดวกและเหมาะสมตามความถนัดดังนี้ครับ:

---

## วิธีที่ 1: ติดตั้ง Node.js และรันโหมดพัฒนา (แนะนำ - สำหรับผู้ใช้ทั่วไป / นักพัฒนา)

วิธีนี้เป็นวิธีมาตรฐานที่ดีที่สุดและเสถียรที่สุดในการใช้งานในเครื่องตัวเอง:

### สิ่งที่ต้องติดตั้งก่อนเริ่มต้น:
1. ดาวน์โหลดและติดตั้ง **Node.js** (แนะนำเวอร์ชัน LTS ล่าสุด) จากเว็บไซต์: https://nodejs.org
2. หลังจากติดตั้งแล้ว เปิด Terminal / Command Prompt ในคอมพิวเตอร์ของคุณ

### ขั้นตอนการรันระบบ:
1. **ดาวน์โหลดไฟล์ของแอปพลิเคชันนี้มาลงคอมพิวเตอร์**:
   - คุณสามารถส่งออกไฟล์เป็น ZIP จากเมนูตั้งค่าของ Google AI Studio ได้เลย
   - แตกไฟล์ ZIP ไว้ในโฟลเดอร์ที่คุณต้องการ (เช่น `C:\projects\holiday-manager`)
2. **เปิดโฟลเดอร์นั้นใน Terminal / Command Prompt**:
   ```bash
   cd path/to/your/folder
   ```
3. **ติดตั้งส่วนประกอบที่จำเป็น (Dependencies)**:
   ```bash
   npm install
   ```
4. **ทำวิธีรันในโหมดใช้งานส่วนตัวบนคอมหน้าจอ (Development Server)**:
   ```bash
   npm run dev
   ```
5. ระบบจะแสดงลิงก์จำลอง เช่น `http://localhost:3000` (หรือเลขพอร์ตอื่นที่ระบุ) บนจอของคุณ คุณสามารถเปิด Browser เข้าลิงก์นี้เพื่อใช้งาน โปรแกรมจะปลอดภัยและบันทึกข้อมูลพนักงานไว้ในเครื่องนั้นอย่างถาวร

---

## วิธีที่ 2: รันผ่านตัวเซิร์ฟเวอร์แบบพกพาโดยตรง (ไม่ต้องโหลด Code ติดตั้งระยะยาว)

หากคอมพิวเตอร์ของคุณมี Node.js ติดตั้งอยู่แล้ว แต่อยากรันเพียงชั่วคราวอย่างรวดเร็ว:

1. เปิด Terminal ในโฟลเดอร์โปรเจกต์
2. รันคำสั่งด้านล่างนี้ได้เลย:
   ```bash
   npx vite
   ```
   *ระบบจะติดตั้งและเปิดหน้าเว็บสำหรับการทำงานหลังบ้านให้ทันทีโดยไม่ต้องรันคำสั่งอื่น*

---

## วิธีที่ 3: เปิดโปรแกรมจากการแปลงไฟล์ Build สำเร็จรูป (รันออฟไลน์โดยสมบูรณ์)

หากคุณติดตั้ง Node.js สำเร็จแล้ว และอยากทดลอง Build ให้เป็นโฟลเดอร์รันเปล่า ๆ ไปวางที่ไหนก็ได้:

1. รันคำสั่งแปลงไฟล์:
   ```bash
   npm run build
   ```
   *(คุณจะได้โฟลเดอร์ที่ชื่อ `dist` ซึ่งบรรจุไฟล์เบาๆ เพียงไม่กี่ไฟล์)*
2. ติดตั้งปลั๊กอินสำหรับรัน Static Server ทั่วไปผ่าน Node.js:
   ```bash
   npx serve dist
   ```
   หรือนำไฟล์ในโฟลเดอร์ `dist` นี้ไปใช้กับโปรแกรมจำลองเซิร์ฟเวอร์ใด ๆ ก็ได้ เช่น XAMPP (โฟลเดอร์ htdocs), IIS, Nginx หรือระบบโฮสติ้งฟรีอื่น ๆ เพื่อเปิดใช้งานครับ!
